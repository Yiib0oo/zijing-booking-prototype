## 三、场景流程图（User Flow）

### 3.1 核心挂号流程（主线）

```
s0 首页 ──[挂号预约]──→ s1 预约须知 ──[已阅读同意]──→ s2 挂号预约
                                                      ├── 找科室 → 子科列表 → s3 选择医生
                                                      └── 找医生 → openS4() → s4 医生主页
                                                                                    │
s3 选择医生 ──[openS4() 点医生]──→ s4 医生主页 ──[选时段弹窗]──→ s5 预约确认
                                                                      │
                                                            [立即预约]
                                                                      ↓
                                                                   s6 挂号支付
                                                                      │
                                              ┌──[取消预约 hist()]───┘
                                              │
                                              └──[openPay() 🔔 payMask]──→ 确认支付 → s7 挂号详情 ✅
```

### 3.2 复诊预约流程

```
s0 首页 ──[复诊预约]──→ s29 复诊预约（显示就诊历史医生列表）
                              │
                    [openS4() 点医生]──→ s4 医生主页
                              │
                    （后续同挂号流程：s4→s5→s6→s7）
```

### 3.3 复诊续方流程

```
s0 首页 ──[复诊续方]──→ s10 处方查询（门诊处方Tab，默认展开最近一次）
                              │
               ┌──────────────┼──────────────┐
               ↓                             ↓
      折叠处方条 1（张明远 06-28）    折叠处方条 2（王建国 05-15）
       ├ ☑ 盐酸舍曲林片 − 1 +         └ ☑ 丙戊酸钠缓释片 − 1 +
       └ ☑ 奥氮平片 − 1 +
               │
               └──[确认续方]──→ 🔔 billMask 支付弹窗
                                    │
                          [确认支付]──→ alert 成功 → switchTab(s0)
```

### 3.4 缴费查询流程

```
s0 首页 ──[缴费查询]──→ s33 缴费查询
                           ├── 未付Tab（当前仅处方缴费¥446.50 一条）
                           │     ├── [支付] → 🔔 billMask
                           │     ├── [查看详情] → s34 缴费详情
                           │     └── [开具发票] → alert
                           │
                           └── 已支付Tab（5条记录）
                                 ├── [查看详情] → s34 缴费详情
                                 └── [开具发票] → alert
```

### 3.5 通知中心 → 各类型页面

```
s0 首页 ──[🔔通知铃]──→ s18 通知中心（6个Tab筛选）
                           │
     ┌──────────┬──────────┼──────────┬──────────┐
     ↓          ↓          ↓          ↓          ↓
   挂号卡片  缴费卡片   检查卡片   取药卡片   就诊卡片
     ↓          ↓          ↓          ↓          ↓
    s7      见下方      s20        s22       s12
 挂号详情
               │
     ┌────┬────┼────┬────┐
     ↓    ↓    ↓    ↓    ↓
   挂号  处方   检验  检查   通用
   缴费  缴费   缴费  缴费   缴费
     ↓    ↓    ↓    ↓    ↓
    s7  s10  s20  s20  s20
```

### 3.6 诊疗资料 → 报告查看

```
s9 诊疗资料 ──Tab──→ 检验报告 / 检查报告
                         │
                    点已出结果卡片 → openReport() → s31 报告查看（PDF预览页）
                         │
                     点未出结果 → alert("报告尚未出具")
```

### 3.7 就诊人管理流程

```
s99 我的
  ├── my-top（头像+手机号）──→ s32 账号设置
  ├── [切换就诊人] ──→ s27 添加就诊人（就诊人列表）
  │       ├── 点就诊人 → switchPatient() → 切为当前 → hist() 返回
  │       ├── [编辑] → editPatient(i) → s28 就诊人管理（表单预填）
  │       └── [+ 添加] → s28 就诊人管理（空表单）
  └── [添加就诊人] ──→ s28 就诊人管理（空表单）

s28 就诊人管理
  └── [添加] 或 [编辑保存] → submitAddPatient() → push 或 update → hist() 返回
```

### 3.8 设置 → 账号设置

```
s99 我的 ──[设置]──→ s26 设置
  │                    ├── 系统设置：字体大小 / 语言
  │                    ├── 账号设置：账号与安全 → s32 / 隐私协议 / 用户协议 → alert
  │                    ├── 其他：关于 / 清除缓存 → alert
  │                    └── [退出登录] → switchTab(s0)
  │
  └── my-top(账号与安全)──→ s32 账号设置
                               ├── 修改手机 / 修改密码 / 隐私 / 用户 → alert
                               └── [退出登录] → switchTab(s0)
```
